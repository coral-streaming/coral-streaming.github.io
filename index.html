Sources

Use just a single API to let you ingest data from multiple data sources. Leverage millions of events created daily across numerous data sources, both internally and (if required) externally. Fast, secure and robust.

Actions

Extract high quality insight, you need to focus on only the relevant data. Addressing this challenge, our platform takes you far beyond simple keyword searches allowing you to filter data with precision.

Sinks

Use our robust data connectors to deliver data to a variety of data destinations. REST and Streaming http connectors, but also a set of push connectors for other systems such as (noql) databases, intranet services, bigdata and logging platforms.

Technologies

Akka is a toolkit and runtime for building highly concurrent, distributed, and resilient message-driven applications on the JVM.

Akka provides a flexible system to instantiate millions of small footprint actors, performing complex event processing in sub-millisecond response times.

Cassandra is a key-value columnar data store. It's distributed and resiliant across multiple nodes, racks and data centers.

Provides a persistency layer for the actors' state. It guarantees that the state of the actor is persisted and can be re-instated if the actor ceases to be life.

Spark provides an advanced DAG execution engine that supports cyclic data flow and in-memory computing. It provides sql, graph, and machine learning computations on distributed data structures in memory.

We use it in order to provide aggregated statistics, model fitting, and advanced patter matching on large streaming datasets.

A language where object-oriented meets functional. Have the best of both worlds. Construct elegant class hierarchies for maximum code reuse and extensibility, implement their behavior using higher-order functions. Or anything in-between.

Scala provides a coincise way for describing the functionality, without being distracted by verbose sintax. It is also provide a very clean constructs to work with failures, exceptions, and multi-threading with very low code complexity

Spray provides a small, embedded and super-fast HTTP server that is a great alternative to servlet containers. spray-can is fully asynchronous and can handle thousands of concurrent connections. It also supports request and response streaming, HTTP pipelining and SSL.

It serves us in order to provide RESTful and streaming http json endpoints in order to expose the streaming analytics services.

Program against your datacenter like it’s a single pool of resources Apache Mesos abstracts CPU, memory, storage, and other compute resources away from machines (physical or virtual), enabling fault-tolerant and elastic distributed systems to easily be built and run effectively.

It provides a well managed, elastic distributed OS and resource allocator, upon which we place our functional components such as spark and akka.
